# Beispielskripte Lernblock Lineare Programmierung

Dieses Verzeichnis enthält Beispielskripte zum Lernblock Lineare Programmierung. Die enthaltenen Skripte dienen der Veranschaulichung und praktischen Anwendung zentraler Konzepte der linearen Optimierung und beziehen sich auf die Übungen des Lernblocks. 

Für alle Übungen, wo Modelle erstellt und gelöst werden, steht ein Jupyter Notebook zur Verfügung, welches die Anwendung der einzelnen Skripte demonstriert und Erklärungen zur Interpretation der Ergebnisse liefert.

**Tipps:** 
- Wenn Sie diese Datei in VSCode lesen und sie hässlich aussieht, liegt das vermutlich daran, dass Sie sie als Textdatei anzeigen lassen. Drücken Sie dann Crtl+Shift+V, um in die Visualisierungs-View zu wechseln.
- Um den Sourcecode hinter einer Funktion im Jupyter Notebook anzuzeigen, drücken Sie Sie in VSCode einfach die Ctrl-Taste und clicken Sie mit dem Mousepointer auf den Funktionsnamen. Danach kehren Sie mit Alt+'<-' (Rückwärtspfeil-Taste) wieder zum zuletzt angezeigten Fenster (d.h. dieses Jupyter Notebook) zurück.
